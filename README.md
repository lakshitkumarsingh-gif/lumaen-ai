# LUMAEN - One Brain. Every AI. Your Knowledge.

## 🚀 Vision
LUMAEN is an adaptive AI assistant platform that learns from your conversations, routes queries to the best AI model (GPT/Claude/Gemini), stores your knowledge in a neural memory graph, and provides deep research capabilities.

## 🎯 Features
- ✅ Google OAuth Login with personalized dashboard
- ✅ Multi-AI Router (GPT/Claude/Gemini intelligent routing)
- ✅ Neural Memory Graph (learns from every conversation)
- ✅ 3 Premium Themes (Dark, Light, Rainbow with glow effects)
- ✅ 4-Tier Pricing Plans
- ✅ Deep Research & Knowledge Spaces
- ✅ Real-time Chat with AI
- ✅ Memory & Context Management

## 🛠️ Stack
**Frontend:** Next.js 14 + TypeScript + Tailwind CSS + Framer Motion
**Backend:** Node.js + Express + PostgreSQL + Redis
**AI:** Multi-LLM Routing System (Ready for OpenAI, Anthropic, Google APIs)
**Auth:** Google OAuth 2.0

## 📦 Project Structure
```
lumaen-ai/

└── docs/                     # Documentation
```

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- PostgreSQL 14+
- Redis 7+
- Google OAuth credentials

### Setup

1. **Clone and Install**
```bash
cd lumaen-ai
npm install
```

2. **Environment Setup**
```bash
cp .env.example .env.local
# Fill in your Google OAuth credentials
```

3. **Start Development**
```bash
npm run dev
```
